# Backend Second-Pilot

Состоит из двух частей _Control_ и _LLM_.

Сontrol будет принимать запросы с frontend'a и обрабатывать их. Отвечает за аутентификацию и авторизацию. Пересылает запросы в LLM. Также он делает запросы на создание базы данных. Для отладки доступен порт `8080`.

LLM будет выполнять работу с языковыми моделями через [OpenRouter](https://openrouter.ai/) и возвращает результат в Control, который вернёт всё на frontend. Для отладки доступен порт `8081`.
