# Backend Second-Pilot

Состоит из двух частей _Control_ и _LLM_.

Сontrol отвечает за аутентификацию, управление пользователями и чатами.

LLM будет выполнять работу с языковыми моделями через [OpenRouter](https://openrouter.ai/) или [LM Studio](https://lmstudio.ai/).
